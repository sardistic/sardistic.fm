> **Personal music analytics visualization dashboard.**

A deep-dive visualization of my listening history, exploring trends, binge habits, and musical identity over the years. Built with a focus on neon cyber-aesthetics and fluid interactivity.

![Dashboard Preview](https://github.com/user-attachments/assets/placeholder-image) *(Screenshot recommended)*

## ‚ú® Features
*   **Total Activity Graph**: Interactive Area Chart showing scrobble density over the last decade.
*   **Year Grid**: "Hero Cards" for each year with:
    *   **Dynamic Glow**: Scale & brightness powered by total play count.
    *   **Waveform Sidebar**: Monthly scrobble distribution visualized as a glowing sidebar.
    *   **Day/Night Split**: Time-of-day breakdown (Sun/Moon icons).
*   **Deep Dives**: Click any year to reveal top artists, albums, and detailed metrics.
*   **Binge Detector**: Algorithms that identify obsessive listening streaks (30+ plays/week).
*   **Glassmorphism UI**: High-performance frosted glass effects with dynamic spotlight borders.

## üöÄ Getting Started

### Prerequisites
*   Node.js (v18+)

### Installation
```bash
# 1. Clone the repository
git clone https://github.com/sardistic/sardistic.fm.git
cd sardistic.fm

# 2. Install dependencies
npm install

# 3. Run development server
npm run dev
```
